# Adult Income Classification

Predicting whether an individual earns more than $50K using demographic and employment data.

## Project overview

This project uses the Adult Income dataset to classify whether an individual is likely to earn more than $50K per year based on demographic and work-related attributes. The project demonstrates data preparation, categorical encoding, feature selection and comparison of classification models.

## Tools used

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook

## Modelling approach

Logistic Regression, K-Nearest Neighbours, Gaussian Naive Bayes, Decision Tree and Random Forest

## Evaluation

Accuracy and confusion matrix analysis

## Key work completed

- Cleaned and explored a dataset with 48,842 observations and 15 features.
- Prepared categorical variables for modelling using encoding methods.
- Used Recursive Feature Elimination to test model performance across different feature sets.
- Compared multiple classification models and discussed overfitting risk between training and testing performance.
- Interpreted confusion matrix results to explain model trade-offs.
