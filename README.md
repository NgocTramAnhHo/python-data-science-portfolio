# Used Car Price Forecasting

Forecasting used car sale prices using feature engineering and ensemble regression models.

## Project overview

This project predicts used car prices from vehicle listing data. It focuses on a practical forecasting problem where accurate predictions can support pricing, inventory and sales decisions for dealerships and individual sellers.

## Tools used

Python, Pandas, NumPy, Scikit-learn, XGBoost, LightGBM, Matplotlib, Seaborn, Jupyter Notebook

## Modelling approach

Random Forest Regressor, XGBoost, LightGBM, Elastic Net and Stacking Regressor

## Evaluation

MAPE, with model improvement from 0.12670 to 0.11602 in the final modelling process

## Key work completed

- Worked with training and testing datasets containing 7,000 and 3,000 records respectively.
- Performed feature engineering on variables such as listed date, power, torque and vehicle age.
- Used OLS and Random Forest feature importance to select meaningful predictors.
- Built and compared regression models including Random Forest, XGBoost, LightGBM, Elastic Net and Stacking Regressor.
- Saved the best model using pickling and documented improvement areas for further model development.

Relevant to the Cochlear internship because it is a forecasting project with clear commercial use. It demonstrates predictive modelling, feature engineering, model tuning and business interpretation, which directly aligns with forecasting sales from prospect data.
